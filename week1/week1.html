Notes-Exercises

  • Story Editor (Section 3: Remembering User's Info)
  • If you want to save information in the user's local storage, you need to use "localStorage"; you can keep the information the user enters or relevant information your need. Save information with the next code:
    localStorage.setItem(key, value)
    Retrieve information with the next code:
    var myVariable = localStorage.getItem(key)
    "Key" is the word in which we can find what we saved, and "value" is what we saved ( it could be an integer, double, string, and so on).
  • Master mobile UX: When we design a website, we must start with the mobile design. We should be there, helpful, and quick for our clients. We will consider the place of our content and if the user would understand it. We must be careful when we need to put advertising; these should keep simple, and we have to avoid using Pop-ups.
