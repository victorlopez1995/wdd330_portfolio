It is the principal information that I didn't know about JavaScript and its resources:

  • If a variable defined as "const" has a non-primitive data type, it can change its value.
  • "Symbol" is a kind of variable used to create unique "values" and is a primitive value.
  • You can check if a "value" is a number with the following method: "Number.isFinite(myVariable)".
  • To convert a String to a number with "Number('myNumber')", and a number to a string with "String(3)".
  • Principal methods with arrays that I need to learn:
    • array.length (we can mutate this value).
    • array.pop(): remove last item and return it.
    • array.shift(): remove first element.
    • array.push('element'): append a new element to the end, and return the new length.
    • array.unshift('element'): append a new element to the beginning.
    • array.concat(anotherArray): merge an array with others. This creates a new array.
    • The "spread" operator (...) can be used to merge the elements of two arrays.
    • array.join(): turn the array into a string with all the items separated by commas.
    • array.reverse(): reverse the array.
    • array,sort(): sort the array.
    • array.indexOf('element'): return the index of the element.
    • array.includes('element'): check if the element is in the array
  • The ternary operator is used to validate a condition: "condition ? code to run if the condition is true : code to run if the condition is false;
