  • Objects
    • Ways to create an object:
      const myObject={};
      const myObject= new Object();
    • You can use JavaScript code in the name and the value of the property.
    • Check if properties or methods exist:
      myObject.hasOwnProperty(property)
      Object.keys(myObject)->return an array of all the keys.
      Object.values(myObject)-> return an array of all the values.
      Object.entries(myObject)->return an array of key-value pairs
    • Adding properties: myObject.property="value".
    • Removing properties: delete myObject.property.
    • The date object: const today = new Date(). principal methods:
      • myDate.getDay(); -> integer of the day of the week.
      • myDate.getDate();-> Day of the month.
      • myDate.getMonth();-> get the month (0-11).
      • myDate.getFullYear();
      • myDate.getTime(); get the date in timestamp.
      • myDAte.getTimezoneOffset();-> difference, in minutes, between the local time on the computer and UTC.
      • (you can use "set" instead of "get" with the methods above)
  • "Document Object Model" (DOM) allows you to access all elements on a web page. This represents an HTML document as a network of connected nodes.
    Access to an element:
    • document.getElementByTagName(element);
    • document.getElementByClassName(element);
    • document.querySelector('element');
    • document.querySelectorAll('element');
    Gettings and setting attributes:
    • myElement.getAttribute('attribute');
    • myElement.setAtrtibute('attribute', 'new value');
    Create an Element:
    • document.createElement(element);
    • document.textContent = ' ';
    Adding elements to the page: parent.appendChild (child);
    innerHTML: return the string of an element.
  • Event: events provide the link between the web page and user interactions.
    Event listener: alert when something happens, or an event happens.
    document.body.addEventListener("click", doSomething);
    the "addEventListener" method can also be called without a node and applied to a global object.
    The target property returns a reference to the node that fired that event.
    Get coordinates of an event:
    • screenx, screeny
    • clientx, clienty
    • pagex, pagey
  • List of principal events:
    • Mouse events: click, mousedown, mouseup, dblclick, mouseover, mouseout, mousemove.
    • Keyboard events: keydown, keypress, keyup, shiftkey, ctrlkey, altkey, metakey.
    • Touch events: touchstart, touchend, touchmove,touchenter, touchleave, touchcancel, touches
    Remove event listeners: removeEventListener(); Stop default behavior: preventDefault(); Stop the bubbling phases: event.stopPropagation();
